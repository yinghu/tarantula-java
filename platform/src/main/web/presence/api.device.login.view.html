This HTTP post request can be used to login with a unique device id

HTTP Method : POST /user/action
Header : Accept : application/json
Header : Content-type : application/x-www-form-urlencoded
Header : Tarantula-tag : index/user
Header : Tarantula-action : onDevice
Header : Tarantula-magic-key : the device id
Header : Tarantula-payload-size : the size of payload
JSON Payload : {deviceId:the device id}
